## Tehnologii Utilizate

- HTML5 (structură semantică)
- CSS3 (Flexbox, Grid, media queries, animații)
- Bootstrap 5 (grid, componente, utilități)
- JavaScript (vanilla JS, manipulare DOM, validare)
- Font Awesome 6 (iconițe)
- Google Fonts (Playfair Display, Raleway)

## Pagini Implementate

1. **Acasă (index.html)** - Prezentare generală, servicii, portofoliu, counter, countdown, testimoniale slider
2. **Servicii (services.html)** - Descriere detaliată servicii, accordion, include/nu include
3. **Portofoliu (portfolio.html)** - Galerie cu filtrare, lightbox, animații scroll
4. **Pachete (packages.html)** - Carduri comparative cu tabs, countdown ofertă specială
5. **Testimoniale (testimonials.html)** - Grid testimoniale, slider, video embed
6. **Contact (contact.html)** - Formular cerere ofertă cu validare completă, hartă interactivă

## Funcționalități JavaScript

- ✓ Validare formular cerere ofertă
- ✓ Meniu responsive (hamburger)
- ✓ Slider portofoliu și testimoniale
- ✓ Smooth scroll la toate ancorele
